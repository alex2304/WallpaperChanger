;Для создания новой конфигурации можете скопировать содержимое этого файла в файл с расширением *.ini и по мере создания своей конфигурации удалять прочитанные комментарии (комментарии - строки, начинающиеся с символа "точка с запятой")

[MainSettings]
name="Пейзажи"
; Название конфигурации
icon=land
; Название иконки из списка доступных (если запилю граф. интерфейс, иконку можно будет выбрать через него. пока что нет смысла перечислять имена всех иконок, которые есть в ресурс-файле.). Параметр можно пропустить.
time=60
; Время между сменой обоев (в секундах)
folder=landscapes
; Папка с обоями
subfolders=0
; Учитывать ли подпапки: 0 - не учитывать, 1 - учитывать 
type=timelapse
; Тип: timelapse - папки зависят от времени суток (обязателен раздел timelapse); single - одна папка всё время
[Timelapse]
; Указанные ниже папки должны находиться в папке, указанной в folder
day_folder=day
; Название папки с "дневными" картинками
evening_folder=evening
; Название папки с "вечерними" (и утренними тоже) картинками
night_folder=night
; Название папки с "ночными" картинками
